# Next.js - Socket.IO を用いたリアルタイムチャットアプリ

## クライアント(フロント)
yarn create next-app

yarn add socket.io-client

### 実行
yarn dev

## サーバー(バック)
npm init -y

npm i express nodemon socket.io

### 実行
npm start
